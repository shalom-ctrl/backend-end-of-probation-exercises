Explain Polymorphism and how You've used it in this context:

Polymorphism could be either overloading or overriding.
Overloading refers to creating different implementations for a method while overriding is when a derived class 
rewrites or changes a virtual member of a base class.

In this Project:
Polymorphism was used to create different derived classes from the base class user and implement the SystemManager
which helps a user create different types of records.
